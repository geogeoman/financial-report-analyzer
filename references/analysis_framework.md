# 财报分析标准框架

在进行财报分析时，应遵循以下逻辑步骤以确保分析的全面性和深度。

## 1. 概览与业绩摘要
- 提取年度营业收入、净利润及其同比增长率。
- 关注非经常性损益对利润的影响。
- **分部收入拆解**: 分析各业务分部的收入规模、增速、收入占比和毛利率，识别增长引擎和拖累因素。
- **运营指标**: 提取 MAU、出货量、交付量等核心运营数据，与财务指标交叉验证。

## 2. 盈利质量分析
- **毛利率趋势**: 对比近三年毛利率，分析是成本下降还是售价提升。分部毛利率梯度分析，识别高毛利和低毛利业务的驱动因素。
- **费用控制**: 观察销售费用、管理费用、研发费用的增速是否超过营收增速，计算费用率并评估投入产出比。
- **非经常性损益影响**: 量化公允价值变动、资产处置等非经常项目对利润的扰动程度。对比扣非净利润增速和归母净利润增速，判断核心业务真实盈利能力。

## 3. 资产负债表健康度
- **资产结构**: 关注货币资金、应收账款、存货、固定资产的占比。
- **负债风险**: 检查有息负债规模及偿债压力。计算资产负债率、流动比率、速动比率，与行业健康参考值对比。
- **运营效率**: 计算应收账款周转率（回款效率）、存货周转率（库存管理）、总资产周转率（资产使用效率）。

## 4. 现金流验证
- 经营活动现金流是否持续为正，净现比（经营CF/净利润）是否 > 1.0。
- 投资活动现金流结构与公司战略是否匹配（扩张期应净流出）。
- 筹资活动现金流方向（融资/还债/回购）反映财务策略。
- 自由现金流是否足以覆盖资本开支和股东回报。

## 5. BP视角决策框架

### 5.1 业务组合 BCG 象限定位
将各业务线按毛利率和收入增速分为四个象限:
- **高毛利·高增长**: 明星业务 — 扩张策略，加大投入抢占市场
- **高毛利·稳定增长**: 现金牛 — 维持/收割策略，最大化现金流
- **低毛利·高增长**: 培育业务 — 优化成本结构，提升盈利能力
- **低毛利·低增长**: 瘦狗业务 — 评估战略价值，考虑优化或收缩

### 5.2 运营红线管理
对关键指标设定预警阈值:
- 应收账款周转率低于阈值 → 触发信用政策收紧
- 存货周转率下降超 X% → 触发库存消化计划
- 资本开支占收入比季度环比增幅超 Xpp → 触发上会审批
- 净现比 < 0.8 → 触发利润质量审查

### 5.3 资金规划路线图
- **短期 (1-2季度)**: 保经营现金流底线
- **中期 (全年)**: 资本开支控制在预算区间内
- **长期 (1-3年)**: 新业务实现正向经营现金流，自给自足

### 5.4 BP追问模板
针对关键财务异常自动生成追问:
- **毛利率下降**: 是成本推动还是价格竞争？改善路径是什么？建议追踪指标。
- **现金流 < 净利润**: 利润含金量评估。应收账款、存货质量如何？是否存在激进确认收入？
- **资本开支高增**: 投资回报周期多长？能否匹配收入增长预期？需追踪产能利用率。
- **费用增速超收入增速**: 获客成本是否上升？转化率能否同步提升？建议追踪单位获客成本和留存率。

## 6. 业务与行业地位
- 主营业务的行业竞争力。
- 研发投入及其转化效果。
- 市占率变化趋势。
