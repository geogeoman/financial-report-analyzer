# 核心财务指标参考指南

本参考资料定义了财报分析中常用的核心指标及其计算公式。

| 指标类别 | 指标名称 | 计算公式 | 意义 |
| :--- | :--- | :--- | :--- |
| **盈利能力** | 毛利率 | (营业收入 - 营业成本) / 营业收入 | 衡量产品竞争力和盈利空间 |
| | 营业利润率 | 经营利润 / 营业收入 | 衡量主营业务经营效率 |
| | 净利率 | 净利润 / 营业收入 | 衡量公司最终获利能力 |
| | ROE (净资产收益率) | 净利润 / 平均净资产 | 衡量股东权益的投资回报率 |
| | 费用率 (销售/管理/研发) | 各项费用 / 营业收入 | 衡量费用控制能力 |
| **偿债能力** | 资产负债率 | 总负债 / 总资产 | 衡量公司财务杠杆和风险水平 |
| | 流动比率 | 流动资产 / 流动负债 | 衡量短期偿债能力 |
| | 速动比率 | (货币资金 + 应收账款) / 流动负债 | 衡量即刻偿债能力 (更严格) |
| **营运能力** | 应收账款周转率 | 营业收入 / 平均应收账款余额 | 衡量回款效率 |
| | 存货周转率 | 营业成本 / 平均存货余额 | 衡量库存管理效率 |
| | 总资产周转率 | 营业收入 / 平均总资产 | 衡量资产使用效率 |
| **现金流质量** | 净现比 | 经营活动现金流净额 / 净利润 | 衡量利润的含金量 (> 1.0 为健康) |
| | 自由现金流 | 经营活动现金流 + 资本开支 (通常为负) | 衡量可自由支配的现金 |
| | 资金缺口 | 经营活动现金流 + 投资活动现金流 | 判断经营造血是否覆盖投资消耗 |
| | 资本开支占收入比 | 资本开支 / 营业收入 | 衡量投资强度 (科技公司通常 10%-20%) |
| **运营指标** | MAU | 月活跃用户数 | 平台活跃度 |
| | 出货量/交付量 | 产品出货或交付数量 | 业务规模 |
| | 市占率 | 公司出货量 / 行业总出货量 | 行业竞争力 |
